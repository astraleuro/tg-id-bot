# Telegram ID bot

Данный бот применяется для сообщения пользователю его ID. Используется в информациныых системах для организации рассылки кодов подтверждения простой электронной подписи (ПЭП)


## Оглавление

- [Установка](#установка)
- [Использование](#использование)

## Установка

1. Создать файл .env содержащий переменную BOT_TOKEN с присвоенным токеном, полученным от бота @BotFather.
2. Собрать и запустить контейнер.
```bash
echo "BOT_TOKEN=1234567890:AAFxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx" > .env

### Или скопировать .env.example и вставить токен своего бота.

mv .env.example .env
nano .env

docker compose up -d --build
```

## Использование 

Найти бота в Telegram, при нажатии кнопки "Старт" (отправке команды `/start`) бот должен выдать сообщение "Ваш ID — хххххххххх
Сообщите его администратору"
